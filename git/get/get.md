## Get projects from the repository (獲取項目)	[Back](./../summary.md)

### 1. Create a new project
- Initialize a project

```git
$ git init
```

- Add file into the project

```git
$ git add *
$ git add README
``` 

### 2. Clone a existed project
- Clone a project

```git
$ git clone [url]
```
