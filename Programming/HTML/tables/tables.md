## Tables [Back](./../HTML.md)

- a table represent information in a **grid** format.
- each block in the grid is referred to as a **table cell**.

### 1. Basic structure
- ```<table>```is used to create a table
- ```<tr>```represents each row
- ```<td>```represents each element in this row
	
	```html
<table>
	<tr>
		<td>15</td>
		<td>20</td>
	</tr>
	<tr>
		<td>25</td>
		<td>30</td>
	</tr>
</table>
```
	<table>
	<tr>
		<td>15</td>
		<td>20</td>
	</tr>
	<tr>
		<td>25</td>
		<td>30</td>
	</tr>
</table>
