### Three main properties of Object-oriented Programming
- Inheritance(繼承)
- Encapsulation(封裝)
- Polymorphism(多態)

### Difference between C++ and Java
C++|Java
:----:|:-----:
編譯執行|解釋執行
兼容有C的面向過程|完全面向對象



### JVM
- JVM(Java Virtual Machine): 屏蔽底層接口為上層提供統一接口, 從而實現平臺無關性

### Properties of Java
- 多線程性
- 健壯性(Garbage Collection, Exception, Strict Type Check)
- 動態性
- 安全性
- 可移植性(平臺無關性)
