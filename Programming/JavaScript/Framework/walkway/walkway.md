## Walkway [Back](./../Framework.md)

- **Walkway.js** is a light weight js framework for drawing svg.

###Code

```js
var svg = new Walkway({
	selector: '#path',
	duration: '2000',
	easing: function(t){
		return t * t;
	}
});

svg.draw();

```
