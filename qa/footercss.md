## How to write footer with CSS [Back](./qa.md)

```html
<body>
    <div class="container"></div>
    <div id="footer"></div>
</body>
```

```css
/**
 * html
 */
html {
    height: 100%;
}

/**
 * body
 */
body {
    min-height: 100%;
    height: auto !important;
    height: 100%;
    width: 100%;
    position: relative;
}

/**
 * container
 */
.container {
    padding-bottom: 97px;
}
```
