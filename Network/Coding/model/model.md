## Model of Server [Back](./../Coding.md)

- 服務器分類:
	- 循環服務器: 同一時刻只能處理一個客戶端請求
	- 併發服務器: 同一時刻可處理多個客戶端請求
	- UDP/TCP服務器模型: 
		- UDP: 採用**循环服务器**模型
		- TCP: 採用**併發服務器**模型
